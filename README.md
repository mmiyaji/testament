# Testament

Tournament generator built with React, TypeScript, and Vite.

## Features

- Single elimination
- Double elimination
- Round robin
- Swiss system
- Endless pairing mode
- Japanese and English UI
- Playwright end-to-end coverage

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run lint
npm run build
npm run test
```

## Deployment

The app is configured for GitHub Pages under `/testament/`.
